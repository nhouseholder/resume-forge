- **Worktree edit paths:** Earlier in the session, edits accidentally targeted the main repo path instead of the worktree. Always use the full worktree path when editing inside a worktree. The main repo files were reverted before merge.
- **`detectedField` type:** The store exposes `detectedField` as `string | null`, not `FieldCategory | null`. Cast as `detectedField as FieldCategory` where `FieldCategory` is expected. Fixed in `ResumeEditor.tsx` with `?? null` coercion.
- **AI polish not live-tested:** The Cloudflare Function was built and passes `npm run lint && npm run test && npm run build`, but the Workers AI binding requires Wrangler dev or a live deploy to test. The function logic mirrors `enhance-resume.ts` which is confirmed working.
